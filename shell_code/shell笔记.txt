文档合并
paste a.txt b.txt >c.txt
查询b文件中不包含a文件的内容
cat b|grep -v -f a
diff a.txt b.txt |grep '<'|cut -d ' ' -f 2
tr用法
1)大小写替换
cat a.txt|tr 'a-z' 'A-Z'
2)删除字符或符合
cat a.txt|tr -d '\n'
删除c语言注释
sed '/\/\/.*/d'|sed '/\/\*.*\*\//d'|sed '/\/\*/,/.*\//d' 
正则表达式
^       以xx开头
$       以xx结尾
[a-z]   表示a-z的任意字符
[^a-z]  表示非a-z的开头的字符
.       表示任意字符
*       表示0或者任意个字符
？      表示0个或1个
+       一次或多次
\{m,n\} 重复m到n次
